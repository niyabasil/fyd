# Fidelity Design System

Welcome to the Fidelity Design System, a comprehensive guide to building consistent and high-quality user experiences.

## Theme Overview

This system utilizes a **light** color mode, designed for clarity and a clean aesthetic. The visual style, reflecting "fidelity," is modern and reliable.

## Typography

The entire typography system, including headlines, body text, and labels, consistently uses the **Inter** font family. This choice ensures readability and a cohesive textual experience across all components.

## Shape and Spacing

UI elements feature a **moderate** level of roundedness (value `2`), striking a balance between sharp angles and overly soft curves. The spacing within and between components is set to **normal** (value `2`), providing adequate breathing room without feeling sparse or cramped.

## Color Palette

The core color palette is defined as follows:

*   **Primary Color:** `#1275e2` - This vibrant blue is the cornerstone of interactive elements and brand emphasis.
*   **Secondary Color:** `#5f78a3` - A supporting shade, offering subtle contrast and versatility for secondary actions and components.
*   **Tertiary Color:** `#c55b00` - An accent color for highlights, badges, or specific calls to attention.
*   **Neutral Color:** `#74777f` - A foundational gray for backgrounds, surfaces, and non-chromatic elements, ensuring visual stability.
